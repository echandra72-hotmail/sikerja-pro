# 📊 SiKerja Pro
Aplikasi Manajemen Pekerjaan & Produktivitas untuk Tim Akuntansi.

## Fitur
- Manajemen klien & perusahaan
- Log & rencana pekerjaan (List, Board, Kalender)
- Time tracker per pekerjaan
- Subtasks & komentar
- Absensi otomatis
- Docs / catatan per klien
- Notifikasi deadline & urgent

## Tech Stack
- Frontend : HTML + CSS + Vanilla JavaScript
- Database : Supabase (PostgreSQL)
- Hosting  : Vercel
