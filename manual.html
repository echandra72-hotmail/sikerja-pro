Manual Pengguna SiKerja Pro

Sistem Manajemen Pekerjaan & Produktivitas Tim Akuntansi

Versi 1.0

1 Login

1

Buka aplikasi SiKerja Pro di browser

2

Pilih nama Anda dari dropdown "Pilih User"

3

Masukkan PIN jika ada — kosongkan jika tidak memakai PIN

4

Klik tombol "Masuk →"

5

Sistem otomatis mencatat absensi masuk Anda hari ini

⚠️ Jika dropdown kosong atau muncul pesan error koneksi, hubungi Admin untuk memastikan data user sudah tersedia.

3 Klien Admin Only

⛔ Menu ini hanya dapat diakses dan dilihat oleh Admin. User dengan role Asisten tidak akan melihat menu Klien di sidebar.

Menambah Klien Baru

1

Klik tombol "+ Tambah Klien"

2

Isi form: Nama Perusahaan (wajib), NPWP, Bidang Usaha, PIC, No. Kontak, Catatan

3

Klik "💾 Simpan"

Fitur Lainnya

FiturCara Penggunaan
🔍 Cari KlienKetik nama, NPWP, atau PIC di kolom pencarian
✏️ Edit KlienKlik tombol ✏️ pada baris klien
🗑️ Hapus KlienKlik tombol 🗑️ pada baris klien
📄 Jumlah BarisPilih 10 / 15 / 20 baris di bagian bawah tabel

4 Pekerjaan

Halaman untuk mencatat log pekerjaan yang sudah dilakukan dan merencanakan pekerjaan mendatang.

Menambah Pekerjaan

1

Klik tombol "+ Tambah"

2

Isi Judul dan pilih Klien (keduanya wajib diisi)

3

Isi Kategori, Penanggung Jawab, Asisten, Status, Prioritas, Tanggal, Deadline, dan Catatan sesuai kebutuhan

4

Klik "💾 Simpan"

Pilihan Tampilan

☰
List

Tabel detail semua pekerjaan

⊞
Board

Kanban — drag & drop antar status

📅
Kalender

Lihat pekerjaan per tanggal

Filter & Pencarian

Gunakan filter di atas tabel untuk menyaring berdasarkan Klien, Status, Prioritas, atau User. Pilih jumlah baris 10 / 15 / 20 di bawah tabel.

💡 Pekerjaan yang terkunci ditandai ikon 🔒 — artinya Anda tidak memiliki hak untuk mengubah atau menghapusnya.

5 Detail Pekerjaan

Klik baris pekerjaan mana saja untuk membuka panel detail di sebelah kanan layar.

FiturFungsi
⏱️ Time TrackerKlik ▶ Mulai untuk menghitung waktu kerja, ⏹ Stop untuk menyimpan. Riwayat waktu tercatat otomatis.
☑️ SubtasksTambah dan centang subtask pekerjaan. Hapus subtask dengan tombol ✕.
💬 KomentarTulis catatan atau diskusi. Tekan Enter atau klik Kirim. Semua user bisa membaca dan menulis komentar.
🔄 Ubah StatusKlik untuk berganti status: Belum Dimulai → Dalam Proses → Selesai.

6 Docs

Buat dan simpan catatan atau dokumen bebas, bisa dikaitkan ke klien tertentu.

1

Klik "+ Baru" untuk membuat dokumen baru

2

Isi judul dan isi dokumen

3

Pilih klien terkait (opsional)

4

Klik "💾 Simpan"

💡 Klik dokumen yang sudah ada untuk membuka dan mengedit kembali.

7 Absensi

JenisKeterangan
🟢 OtomatisDicatat saat user berhasil login setiap hari
✍️ ManualKlik "+ Manual", pilih user, tanggal, dan jam masuk

Riwayat absensi ditampilkan secara berurutan dari tanggal terbaru.

8 Manajemen User Admin Only

⛔ Menu ini hanya dapat diakses dan dilihat oleh Admin.

Menambah User Baru

1

Klik "+ Tambah User"

2

Isi Nama, pilih Role, dan PIN (opsional)

3

Klik "💾 Simpan"

Role User

RoleAkses
Admin / AkuntanAkses penuh ke semua menu, data klien, manajemen user, dan seluruh pekerjaan
AsistenAkses terbatas — tidak melihat menu Klien & Users, hanya bisa edit pekerjaan milik sendiri
